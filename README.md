# petty

Generate petty names.

## Usage

```clojure
(require '[petty.core :as petty])

(petty/slug) ; => "beautiful-horse"
(petty/alliteration) ; => "annoyed-anteater"
(petty/emoji) ; => ["🐄" "elegant-cow"]
```
